# OMNI-CORE (On-chain Multi-agent Neural Interface)

An advanced UI/UX for a decentralized AI platform. Built with React, Vite, Tailwind CSS, and Framer Motion for a premium, futuristic experience.

## Features

- **Consciousness Stream**: Real-time AI thought simulation.
- **Delegation Matrix**: 3D visualization of network nodes using React Three Fiber.
- **Live Task Queue**: Interactive dashboard for managing AI tasks.
- **Validation Rubric**: Multi-agent consensus mechanism display.
- **Web3 Integration**: Wallet connection and treasury monitoring.
- **Premium Design**: Dark mode, glassmorphism, and smooth animations.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Structure

- `src/components`: Contains all UI components (Header, Panel, Charts, etc.).
- `src/utils`: Helper functions and utilities.
- `src/App.tsx`: Main application logic.
- `src/main.tsx`: Entry point.

## Deployment to Vercel

1.  Push this repository to GitHub.
2.  Login to Vercel and import the project.
3.  Vercel will automatically detect the Vite settings.
4.  Click **Deploy**.
